# All you need to do:
Open your terminal and write the next lines
1. git clone https://github.com/vitomania/telegram_bot.git
2. cd telegram_bot
3. create _config.json_
```python
{
    "TOKEN": "YOUR TOKEN HERE",
    "dir_name": "data",
    "file_name": "results",
    "write_mode": "a+"
}
```
4. python3 bot.py

You can change the write\_mode: "w+" or "a+" in _config.json_
